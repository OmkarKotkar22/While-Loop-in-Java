## ðŸ”„ While Loop Programs

This section focuses on iterative programming, using the **`while` loop** to repeat a block of code until a specific condition is no longer true.

### Key Concepts
* **Indefinite Iteration:** Repeating a process where the number of iterations is unknown beforehand (e.g., when manipulating digits).
* **Modulus and Division:** Using `%` and `/` inside loops for digit-by-digit processing.
* **Series Generation:** Generating sequences of numbers.

### 1. Number Properties & Checks
These files implement algorithms to check for specific mathematical properties, often requiring a loop:

| File Name | Description |
| :--- | :--- |
| `check_prime_no.java` | Checks if a single number is prime. |
| `primenum1_to_n.java` | Prints all prime numbers in a range (1 to N). |
| `check_perfect_no.java` | Checks for a Perfect number. |
| `check_neon_no.java` | Checks for a Neon number. |
| `check_happy_no.java` | Checks for a Happy number. |
| `harshadno1_to_n.java` | Checks and prints Harshad numbers in a range. |
| `no_palindrome.java` | Checks if a number is a palindrome. |
| `armstrongno1_to_n.java` | Checks and prints Armstrong numbers in a range. |
| `hcf_of_two_no.java` | Finds the Highest Common Factor (HCF) of two numbers. |

## ðŸ’» Getting Started

### Prerequisites
* **Java Development Kit (JDK 8+)** installed.

### Running the Code
1.  Clone this repository:
    ```bash
    git clone [Your Repository URL]
    cd [repository-name]
    ```
2.  Navigate to the program's directory (if structured in sub-folders) or remain in the root directory.
3.  Compile any Java source file (e.g., for `DM_check_grade.java`):
    ```bash
    javac DM_check_grade.java
    ```
4.  Execute the compiled class:
    ```bash
    java DM_check_grade
    ```
